# Release Title

`v0.1.0 - First public Windows release`

# Release Summary

OAuth Account Switcher is a Windows desktop app that helps you switch to the best available Codex account before reopening Codex.

## Highlights

- Save the current Codex auth state as a reusable account card
- Compare 5-hour and 1-week quota windows
- Switch to the recommended account before restarting Codex
- Refresh or delete saved account bundles from one dashboard

## Download

- `OAuth Account Switcher-<version>-x64.exe`: installer
- `OAuth Account Switcher-<version>-x64.zip`: portable build

## Notes

- Windows may show a SmartScreen warning because the app is not code-signed yet.
- The app is currently Windows-only.
- Automatic updates are not included in this release.
